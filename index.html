BIBLIOTECA MÁGICA

Tecnologias: HTML, CSS, JavaScript | Ambiente: VSCode

Esse projeto foi desenvolvido com o objetivo de criar uma biblioteca digital acessível e funcional, reunindo grandes clássicos da literatura brasileira e internacional em um só lugar. 📖✨

A estrutura foi montada usando HTML para a organização dos conteúdos, CSS para estilização e JavaScript para funcionalidades interativas, como a navegação entre categorias e filtros por autor ou gênero. A ideia é oferecer ao usuário uma experiência fluida e visualmente agradável, mesmo sendo um projeto leve e direto.

O design é minimalista, com foco na leitura e usabilidade, utilizando tons suaves que facilitam a concentração. Foi pensado para ser usado tanto em desktops quanto em dispositivos móveis, o que garante acessibilidade e praticidade.

Além disso, o projeto incentiva a leitura gratuita e o acesso à cultura, funcionando como uma espécie de “estante virtual” com obras públicas ou de domínio público, como Dom Casmurro, O Alienista, Orgulho e Preconceito, entre outros títulos marcantes.

🔗 Uma ótima solução para estudantes, professores ou apaixonados por literatura que buscam praticidade e organização ao acessar grandes obras clássicas.

VOCÊ TEM MEMÓRIA BOA? DESCUBRA AGORA!

Tecnologias: HTML, CSS, JavaScript | Ambiente: VSCode

Esse projeto foi pensado para transformar o jeito de estudar em algo mais leve, dinâmico e eficiente. Trata-se de um jogo de flashcards 🃏 criado no VSCode, que permite ao usuário revisar conteúdos de forma divertida e prática, como em um quiz de perguntas e respostas.

A interface foi desenvolvida com foco em simplicidade, permitindo que qualquer pessoa, independentemente da idade ou familiaridade com tecnologia, consiga usar. O usuário pode escolher entre diferentes temas (como matemática, história, inglês etc.), e ao clicar em cada carta, vê a pergunta e depois a resposta. Um contador de acertos e erros permite acompanhar o desempenho ao longo do jogo. 📊

O JavaScript entra como motor das funcionalidades, como embaralhar os cards, contabilizar o progresso e exibir feedback visual. A responsividade garante uma boa navegação tanto em smartphones quanto no computador.

🎯 Ideal para alunos se preparando para provas, vestibulares ou concursos, o projeto também serve como base para futuras melhorias, como a adição de um sistema de login, rankings ou até mesmo um painel de criação de novos flashcards personalizados.

Assista FILMES e SÉRIES DE GRAÇA!

Tecnologias: HTML, CSS, JavaScript | Ambiente: VSCode

Esse é um dos projetos mais robustos, com foco total na experiência do usuário que deseja assistir filmes e séries sem complicações. Desenvolvido inteiramente no VSCode, o site oferece uma interface moderna, intuitiva e responsiva, permitindo que o usuário encontre e assista seus conteúdos favoritos com poucos cliques. 🍿💻

Além do apelo visual, o projeto se destaca pela proposta de democratização do entretenimento, trazendo um ambiente que lembra plataformas conhecidas de streaming, mas com navegação simples e sem burocracias.

📺 Um ótimo exemplo de como unir funcionalidade e estética em um projeto que pode ser expandido futuramente para integrar banco de dados, sistema de usuários e recomendações personalizadas.
